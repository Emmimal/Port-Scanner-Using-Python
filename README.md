# Port Scanner Using Python

Educational repository accompanying the article:
https://emitechlogic.com/how-to-build-a-python-port-scanner-from-scratch/

This project walks through building a Python port scanner from a minimal foundational scanner to a professional tool with:
- range input & validation
- multithreading for speed
- banner grabbing / service identification
- logging & CSV output
- practice exercises and troubleshooting notes

**Important:** Only scan machines you own or have explicit permission to test.
